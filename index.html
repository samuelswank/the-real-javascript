Instructions

To type directly with the computer keyboard:

  • For the emphatic letters, type the capital letters
    example: H, S, D, T, Z
  • To add a diacritical mark, type an apostrophe
    example: t', H', d', s', S', T', g', k', h', n' (ou ñ)
    or 2 apostrophes: g'' (ou N)
